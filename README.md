# 🚀 Engenharia de Software com Java

Este repositório contém minha evolução técnica estruturada em **12 etapas**, com mentoria baseada em IA, focada em **Engenharia de Software com Java**. O objetivo não é apenas dominar a sintaxe, mas aplicar padrões de projeto (Design Patterns), gestão de memória e lógica avançada sob a ótica de sistemas escaláveis e de alta performance.

---

### 🛠️ Tecnologias e Ferramentas

* **Linguagem:** Java 21(LTS)
* **Controle de Versão:** Git & GitHub
* **Ambiente:** Terminal (CLI) e VS Code (Fase inicial)
* **Paradigma:** Orientação a Objetos (em progresso)

---

### 🧠 Princípios de Engenharia aplicados

Ao longo deste estudo, sigo os seguintes guias:

1. **Clean Code:** nomes de variáveis semânticos e métodos curtos.
2. **Fail Fast:** Validação de erros o mais cedo possível.
3. **Eficácia de Memória:** Escolha do tipo primitivo correto para cada cenário. 

---

### 📝 Estrutura de Aprendizado

Em cada etapa deste estudos foi validada através de três pilares de avaliação técnica: 

1. **Desafio Prático Progessivo:** Desenvolvimento de uma aplicação real do zero (ex: Calculadoras, Simuladores de mercado e sistemas de Crédito) para aplicar os conceitos da semana.
2. **Desafio de Refatoração:** Exercício focado em *Clean Code*, onde recebo um código mal escrito ("Código de Estagiário") e devo reescrevê-lo aplicando padrões profssionais e performance.
3. **Pergunta de Entrevista:** Simulação de perguntas teóricas de processos seletivos para validar o entendimento dos fundamentos e da arquitetura do Java(JVM, Memória, tipagem). 

---

### 💡 Como ler este repositório

Para ver os desafios de cada etapa, basta entrar na pasta correspondente. Lá você encontrará:

1. O código fonte original (`.java`).
2. Um arquivo `README.md` específico detalhando o problema resolvido e a explicação da refatoração aplicada.

### 📅 Roadmap da Mentoria

#### **Módulo 1: Fundamentos e Ecossistema**

* **[Etapa 1: O Ecossitema Java](./etapa_1_O_Ecossistema_Java/)**
    - JDK vs JRE vs JVM; Compilação via terminal (`javac` e `java`).

* **Etapa 2: Variáveis e Tipos Primitivos**
    - Gestão de memória (Stack); Tipagem; Casting de dados.

* **Etapa 3: Lógica e Operadores**
    - Operadores lógicos; Curto-circuito; Expressões booleanas simplificadas. 

#### **Módulo 2: Controle de Fluxo**

* **Etapa 4: Estrutura de Decisão**
    - Uso profissional de `if`, `else if`, `else` e `sitch case`. Aplicação de Cláusulas de Guarda.

* **Etapa 5: Laços de Repetição**
    - Estruturas `while` e `for`. Diferenças entre interações e controle de fluxo em loops. 

* **Etapa 6: Manipulação de Strings**
    - Imutabilidade, String Pool e performance com `StringBuilder`.

###  Módulo 3: O Coração da Orientação a Objetos

* **Etapa 7: Classes e Objetos**
    - Atributos, métodos e o conceito de instanciação.

* **Etapa 8: Encapsulamento e Construtores**
    - Modificadores de acesso (`private`, `public`) e proteção da integridade dos dados.

* **Etapa 9: Composição vs Herança**
    - Modelagem de sistemas complexos e o princípio de reutilização de código.

###  Módulo 4: Robustez e Organização de Dados

* **Etapa 10: Coleções (Collections)**
    - Trabalhando com `List`, `Set` e `Map`. Qual estrutura escolher para cada cenário?

* **Etapa 11: Tratamento de Erros Exceptions**
    - `try-catch`, exceções checadas e não checadas. Como criar sistemas resilientes.

* **Etapa 12: Testes Unitários**
    - Introdução ao JUnit e a importância de testar a lógica de negócio.
